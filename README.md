# SPOTIFY API CHART STREAMER
SPOTIFY API CHART STREAMER - GET YOUR SONG IN SPOTIFY TOP 50

## Key Features:
▫️ Chart Streaming (Streams Counts Towards Spotify Charts)

▫️ Separate Invalid and Used Accounts (Can see which accounts are not working and which where used already)

▫️ Option To Skip Already Used Account

▫️ Option To Login Via Email or Regular Login

▫️ Can Set Custom Stream Count For Each Song. 

▫️ Can Set Plays Per Account (To not abuse it)

▫️ Can Set Plays Per Song

▫️ Follow/Like - Songs, Artists, Albums and Playlists

▫️ Supports HTTP/SOCKS5 Proxies

▫️ Supports Anti-Captcha and Capsolver

▫️ A Lot Of More Features
